# Log IP Analyzer

A simple Python script that extracts and counts IP addresses from log files to help identify suspicious login attempts.

## Features
- Reads log files
- Extracts IP addresses using regex
- Counts occurrences of each IP
- Useful for detecting brute-force or suspicious activity

## Usage
1. Put your log file in the same folder as `ip_analiz.py` and name it `log.txt`.
2. Run the script:

```bash
python ip_analiz.py
